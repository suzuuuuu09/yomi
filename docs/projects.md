# プロジェクト企画書

## プロジェクト名
「YOMi」(ヨミ)

## コンセプト
「読書」というものを、宇宙というものに昇華させて楽しく読書管理ができるようにする  
また、積読[^1]への罪悪感を「未観測の星」というワクワク感に変え、知識の繋がりを「星座」として表現することで、読書の楽しさを増幅させる。
[^1]: 積読（つんどく）とは、購入した本を読まずに積んでおくこと

## ターゲットユーザー
- 読書好きな人
- 読書を習慣化したい人
- 積読に罪悪感を感じている人
- 読書の知識の繋がりを可視化したい人
- 読書をあまりしない人

## 問題・課題
- 読書管理が面倒で続かない
- 読書の習慣化が難しい
- 「読まなきゃいけない」というのがストレスになってしまう
- 時間が経つとどの本を読んだか、どの本を積んでいるかがわからなくなる
- 読書の知識の繋がりを把握しづらい
- 読書をするきっかけになってほしい

## 解決策
- 本を登録した瞬間を「星の誕生」、読書中を「星のまたたき」、読了を「星座の確定」として演出
- ジャンルや読了時期などのメタデータを解析し、知識の関連性を「光の線」で結ぶ
- 自分の読書遍歴を3D空間で自由に飛び回り、没入感のある振り返り体験を提供

## 価値提案
- 積読を罪悪感ではなく期待感に変える
- 自分の学んできたことが「星空」として見えるようになって、知識の繋がりを実感できる
- 星を輝かせるために読書を続けるモチベーションが生まれる
